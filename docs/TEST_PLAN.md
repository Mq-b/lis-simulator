## 四、TCP 集成测试（无头模式）

使用 `--headless --tcp` 模式，Python 脚本通过 TCP 连接模拟仪器端。
无需物理串口或 com0com，适用于 CI 和开发测试。

```bash
# 启动无头服务器
cargo run -- --headless --tcp 12345

# 运行测试
python tests/test_tcp.py --port 12345
```

|---|--------|------|------|

| 4 | 校验和验证 | ✅ | 错误校验和的帧被拒绝 |
| 5 | H 记录解析 | ✅ | 发送方/消息类型/版本/时间 |
